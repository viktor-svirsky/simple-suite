# M2 Markdown Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the plain `TextEditor` detail view with a markdown-aware experience: the editor renders an `AttributedString` when *not* being edited and swaps to a plain markdown-source `TextEditor` when the user taps to edit. `#tag` mentions and fenced code blocks are styled inside the rendered view.

**Architecture:** A new `MarkdownRenderer` (pure, testable) turns a markdown string into an `AttributedString` by building on `Foundation`'s `AttributedString(markdown:)` parser and then walking the result to apply custom styling for `#tag` tokens and monospaced code. A new `NoteMarkdownView` renders that attributed string. `NoteEditorView` owns an `isEditing` flag and swaps between `NoteMarkdownView` (rendered) and `TextEditor` (source).

**Tech Stack:** SwiftUI, `AttributedString(markdown:)`, `AttributedString.Runs` iteration, regex for tag ranges, `Text(AttributedString)`.

**Branch:** `feat/m2-markdown` off `main` (after M1 is merged). Commit per task. No pushes.

**Reference spec:** `docs/superpowers/specs/2026-04-19-simple-notes-design.md`

## Scope

### In (M2)
- Markdown rendering of note body when not editing — bold, italic, inline code, lists, headings 1-3, links, fenced code blocks
- Tap-to-edit: tapping the rendered view flips to `TextEditor` focused at the tap location (best-effort; caret ends up at end on first pass)
- Tap-outside (end editing) returns to rendered view
- `#tag` tokens styled mono + muted in the rendered view (not tappable yet — filtering ships in M3)
- `NoteListRow` preview pulls plain text (markdown stripped) so the list stays legible
- Markdown cheatsheet sheet reachable from the editor toolbar

### Out (deferred)
- Tapping a `#tag` to filter — M3
- Attachment image rendering (`![](attachment://id)`) — M4
- Syntax highlighting inside code blocks
- Block quotes styling (render as italic-muted is enough for M2)

### Non-goals
- Full WYSIWYG with inline formatting while typing
- Server-side or SwiftSyntax-based markdown — stick to Foundation's built-in parser

- [x] **Step 3: Implement scanner** — NOTE: Used manual scanning instead of regex literal because Swift Regex literals do not support lookbehind (`(?<=...)`). Same semantics: `#` must start the string or follow whitespace/punctuation; name must be non-empty `[A-Za-z0-9_-]+`.

Create `apps/simple-notes/SimpleNotes/Markdown/MarkdownTagScanner.swift`:

```swift
import Foundation

struct MarkdownTag: Equatable {
    let name: String
    let range: Range<String.Index>
}

enum MarkdownTagScanner {
    /// Matches `#` followed by at least one [A-Za-z0-9_-], where the `#` itself
    /// is at the start of the string or follows whitespace/punctuation (so
    /// "C#is" doesn't match but "a #tag" does). Bare `#` (markdown heading) is
    /// skipped because we require a word character immediately after.
    private static let pattern = /(?<=^|[\s(,.;:!?\[\]{}])#([A-Za-z0-9_-]+)/

    static func tags(in text: String) -> [MarkdownTag] {
        text.matches(of: pattern).map { match in
            MarkdownTag(
                name: String(match.output.1),
                range: match.range
            )
        }
    }
}
```

Create `apps/simple-notes/SimpleNotes/Markdown/MarkdownRenderer.swift`:

```swift
import Foundation
import SwiftUI

enum MarkdownRenderer {
    /// Parses `markdown` with Foundation's built-in parser (full-document mode,
    /// so newlines survive as paragraph breaks), then decorates `#tag` runs
    /// with a mono-muted style. Code blocks already come through as
    /// `presentationIntent(.codeBlock)` which iOS renders monospaced by default.
    static func render(_ markdown: String) -> AttributedString {
        let options = AttributedString.MarkdownParsingOptions(
            allowsExtendedAttributes: true,
            interpretedSyntax: .full
        )
        var attributed: AttributedString
        do {
            attributed = try AttributedString(
                markdown: markdown,
                options: options
            )
        } catch {
            attributed = AttributedString(markdown)
        }

        // Tag styling: walk the flattened text, match `#tag` ranges, project
        // them onto AttributedString index space.
        let flat = String(attributed.characters)
        let tags = MarkdownTagScanner.tags(in: flat)
        for tag in tags {
            let lower = flat.distance(from: flat.startIndex, to: tag.range.lowerBound)
            let upper = flat.distance(from: flat.startIndex, to: tag.range.upperBound)
            guard
                let start = attributed.index(
                    attributed.startIndex,
                    offsetByCharacters: lower
                ),
                let end = attributed.index(
                    attributed.startIndex,
                    offsetByCharacters: upper
                )
            else { continue }
            attributed[start..<end].font = .system(.body, design: .monospaced)
            attributed[start..<end].foregroundColor = Theme.Color.muted
        }

        return attributed
    }
}

private extension AttributedString {
    /// Offset helper — returns nil if the offset is out of bounds.
    func index(_ base: AttributedString.Index, offsetByCharacters n: Int) -> AttributedString.Index? {
        var idx = base
        for _ in 0..<n {
            guard idx < endIndex else { return nil }
            idx = characters.index(after: idx)
        }
        return idx
    }
}
```

```swift
import Foundation

enum MarkdownPlainText {
    /// Best-effort reduction of markdown to plain text for list previews. Uses
    /// the Foundation parser then returns its character contents — for the
    /// subset we care about (bold/italic/code/links/list markers) this drops
    /// the syntax automatically.
    static func strip(_ markdown: String) -> String {
        let options = AttributedString.MarkdownParsingOptions(
            allowsExtendedAttributes: false,
            interpretedSyntax: .inlineOnlyPreservingWhitespace
        )
        let attributed = (try? AttributedString(markdown: markdown, options: options))
            ?? AttributedString(markdown)
        return String(attributed.characters)
    }
}
```

```swift
    var preview: String {
```

```swift
import SwiftUI
import SwiftData

struct NoteEditorView: View {
    @Bindable var note: Note
    @Environment(\.modelContext) private var modelContext

    @State private var autosaver = EditorAutosaver()
    @State private var isEditing = false
    @FocusState private var isFocused: Bool
    @State private var showCheatsheet = false

    var body: some View {
        Group {
            if isEditing {
                TextEditor(text: $note.body)
                    .font(Theme.Font.serif(17))
                    .lineSpacing(6)
                    .focused($isFocused)
                    .scrollContentBackground(.hidden)
                    .background(Theme.Color.bg)
                    .padding(.horizontal, Theme.Metric.padding)
                    .padding(.vertical, 8)
                    .onChange(of: note.body) { _, _ in
                        autosaver.scheduleTouch(on: note)
                    }
            } else {
                NoteMarkdownView(markdown: note.body) {
                    beginEditing()
                }
            }
        }
        .navigationTitle(note.title)
        .navigationBarTitleDisplayMode(.inline)
        .onDisappear(perform: flush)
        .toolbar {
            ToolbarItem(placement: .topBarTrailing) {
                Button {
                    note.isPinned.toggle()
                    note.touch()
                } label: {
                    Image(systemName: note.isPinned ? "pin.fill" : "pin")
                }
                .accessibilityLabel(note.isPinned ? "Unpin" : "Pin")
            }
            ToolbarItem(placement: .topBarTrailing) {
                Button {
                    showCheatsheet = true
                } label: {
                    Image(systemName: "questionmark.circle")
                }
                .accessibilityLabel("Markdown cheatsheet")
            }
            if isEditing {
                ToolbarItem(placement: .topBarTrailing) {
                    Button("Done") { endEditing() }
                        .font(Theme.Font.sans(15, weight: .medium))
                }
            }
        }
        .sheet(isPresented: $showCheatsheet) {
            NavigationStack {
                MarkdownCheatsheetView()
                    .toolbar {
                        ToolbarItem(placement: .topBarTrailing) {
                            Button("Close") { showCheatsheet = false }
                        }
                    }
            }
        }
    }

    private func beginEditing() {
        isEditing = true
        isFocused = true
    }

    private func endEditing() {
        isFocused = false
        isEditing = false
        flush()
    }

    private func flush() {
        autosaver.cancel()
        note.touch()
        try? modelContext.save()
    }
}
```
