# M5 iCloud Sync Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wire SwiftData's CloudKit private-database sync behind a build flag. AltStore sideloaded builds (default) stay local-only. TestFlight / App Store builds (with signing + entitlements) get iCloud sync automatically.

**Architecture:**
- New build setting `SIMPLE_NOTES_CLOUDKIT_ENABLED` (0/1) compiled into Swift via `SWIFT_ACTIVE_COMPILATION_CONDITIONS`.
- `ModelContainer` construction switches on this flag: `.private` when enabled, `.none` otherwise.
- All `@Model` types already have defaults + optional relationships (required for CloudKit). Verify and fix any that don't.
- Settings screen shows sync status: "Syncing", "Synced · 2m ago", "Offline", or "Disabled for this build".
- Sync status derived from `NSPersistentCloudKitContainer.eventChangedNotification` (or SwiftData equivalent in iOS 17) — light wrapper that publishes status via `@Observable`.

**Tech Stack:** SwiftData, CloudKit (private DB), entitlements, `NSUbiquityIdentityDidChange` for account status.

**Branch:** `feat/m5-icloud` off `main`.

**Reference spec:** `docs/superpowers/specs/2026-04-19-simple-notes-design.md`

## Scope

### In (M5)
- Build flag gating CloudKit (off by default)
- `.entitlements` file with iCloud container + key-value store
- `project.yml` target entitlements + `SIMPLE_NOTES_CLOUDKIT_ENABLED` setting
- `ModelContainer` factory function (`ModelContainerFactory`) that chooses config
- Model-shape audit: CloudKit requires all attributes to have defaults and all relationships to be optional or have inverse
- `SyncStatus` observable + sidebar footer wiring
- Settings screen with account status + last sync + "Open iCloud Settings" deep link
- Tests: model container factory picks right config, sync status transitions

### Out (deferred)
- Public database sharing
- Share-sheet / collaborative notes
- Per-folder or per-note sync opt-in

### Non-goals
- Manual "Force sync now" button
- Custom conflict resolution UI — rely on SwiftData's last-write-wins

- [ ] **Step 1: Factory**

```swift
import Foundation
import SwiftData

enum ModelContainerFactory {
    static func make() throws -> ModelContainer {
        let schema = Schema([
            Note.self, Folder.self, Tag.self, Attachment.self,
        ])
        let configuration: ModelConfiguration

        #if SIMPLE_NOTES_CLOUDKIT_ENABLED
        configuration = ModelConfiguration(
            "SimpleNotes",
            schema: schema,
            isStoredInMemoryOnly: false,
            cloudKitDatabase: .private("iCloud.com.simplesuite.notes")
        )
        #else
        configuration = ModelConfiguration(
            "SimpleNotes",
            schema: schema,
            isStoredInMemoryOnly: false,
            cloudKitDatabase: .none
        )
        #endif

        return try ModelContainer(for: schema, configurations: [configuration])
    }

    static func inMemory() throws -> ModelContainer {
        try ModelContainer(
            for: Note.self, Folder.self, Tag.self, Attachment.self,
            configurations: ModelConfiguration(isStoredInMemoryOnly: true)
        )
    }
}
```

- [ ] **Step 2: App uses factory**

```swift
@main
struct SimpleNotesApp: App {
    let container: ModelContainer = {
        do { return try ModelContainerFactory.make() } catch { fatalError("\(error)") }

    }
```

## Task 5: Settings screen

**Files:**
- Create: `apps/simple-notes/SimpleNotes/Features/Settings/SettingsView.swift`
- Modify: `apps/simple-notes/SimpleNotes/App/RootView.swift` / sidebar to present it

- [ ] **Step 1: Build settings view**

```swift
struct SettingsView: View {
    @Environment(\.syncStatus) private var sync

    var body: some View {
        List {
            Section("iCloud") {
                LabeledContent("Status", value: statusText)
                if case .disabled = sync.state {
                    Text("This build has iCloud sync disabled. Install from TestFlight to enable.")
                        .font(Theme.Font.sans(13))
                        .foregroundStyle(Theme.Color.muted)
                }
                Button("Open iCloud Settings") {
                    if let url = URL(string: UIApplication.openSettingsURLString) {
                        UIApplication.shared.open(url)
                    }
                }
            }
            Section("About") {
                LabeledContent("Version", value: appVersion)
                LabeledContent("Bundle", value: Bundle.main.bundleIdentifier ?? "—")
            }
        }
        .navigationTitle("Settings")
    }

    private var statusText: String {
        switch sync.state {
        case .disabled: return "Disabled (build flag off)"
        case .offline: return "Offline"
        case .syncing: return "Syncing…"
        case .synced(let date):
            let f = RelativeDateTimeFormatter()
            return "Synced · " + f.localizedString(for: date, relativeTo: Date())
        }
    }

    private var appVersion: String {
        let v = Bundle.main.object(forInfoDictionaryKey: "CFBundleShortVersionString") as? String ?? "?"
        let b = Bundle.main.object(forInfoDictionaryKey: "CFBundleVersion") as? String ?? "?"
        return "\(v) (\(b))"
    }
}
```
